### Prerequisites

- Node.js (v14 or later)
- pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/watchyo.git
   cd watchyo
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```
